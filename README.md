# LOL-record
🕹 롤 전적 검색 사이트, 웹프로그래밍 개인 텀프로젝트 🖲

## 📚 Summary
- 2020학년도 2학기 (4학기) 웹 프로그래밍 과목 개인 텀프로젝트
- 라이엇에서 배포하는 API를 이용하여 유저들의 게임 전적, 현재 게임 중인지 아닌지와 회원정보를 조회할 수 있다!
- 웹의 기본 기능으로 회원가입과 로그인 기능도 구현해뒀다
- 챌린저 티어의 유저를 조회할 수 있다
